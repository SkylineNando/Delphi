TEdit Managed Bindings Samples

Project: SimpleManagedBindEdits

Summary: Creates a managed binding between two TEdit control.

Description: A TBindExpression component is used to create a managed binding.  OnChange events notify the expression engine of a change to one of the TEdit controls.  This may cause re-evaluation of the binding expression(s) and TEdit update.




Project: ManagedBindEdits

Summary: Enhanced version of SimpleManagedBindEdits.

Description: Compared to SimpleManagedBindEdits, this application displays more information and provides runtime control over binding direction.  Remember to check the "Active" checkbox.
