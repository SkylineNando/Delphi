Description Demonstrates use of DBExpress components. This demo requires that paths be specified for the various drivers in dbxconnections.ini; for instance, if you have Interbase server running, you can click the Files tab and view Dbxconnections.ini, you can edit the database file name to C:\Users\Public\Documents\RAD Studio\\Samples\Data\employee.gdb, Ctrl-S to save, then select IBConnection from the "Connection" tab and connect and try out the various functions.
