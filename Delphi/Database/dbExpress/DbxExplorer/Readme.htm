Description This demo shows how to access dbExpress administration files and schema information.

This example requires that Interbase XE be installed. Make sure the server is running before you run the example.
Also you have to specify the full path to the database in the TSQLConnection component: localhost/gds_db:<FULL_DB_PATH>.
