1         TADDataMove – Main

The demo demonstrates the main ways to move data using TADDataMove:

1.)     from text file to dataset

2.)     from one dataset to other dataset

3.)     from dataset to text file

Note! In all listing it’s used sample code with TADDataMove component named dtmMain.

1.)     Moving data from text file to dataset

with dtmMain do begin

  // Set up kind of source

  SourceKind := skText;

  // Set up text file name

  // ADExpandStr utility expands the variables like ‘$(ADHOME)’ to the string

  TextFileName := ADExpandStr('$(ADHOME)\Samples\Comp Layer\TADDataMove\Main\Data.txt');

  // Set delimiter char

  TextDataDef.Separator := ';';

  // If text file contains the field names TextDataDef.WithFieldNames must be True

  TextDataDef.WithFieldNames := True;

  // Use TADQuery as Destination

  Destination := qryLoaded;

end;

dtmMain.Execute;

2.)     Moving data from dataset to dataset

with dtmMain do begin

  // Use TADTable as Source

  Source := qryLoaded;

  // Use TADTable as Destination

  Destination := qryMoved;

  // Setting other properties

  Mode := dmAlwaysInsert;

  Options := Options + [poClearDest];

end;

dtmMain.Execute;

3.)     Moving data from dataset to dataset

with dtmMain do begin

  // Set up text file name

  TextFileName := ADExpandStr('$(ADHOME)\Samples\Comp Layer\TADDataMove\Main\DataOut.txt');

  // Use TADQuery as Source

  Source := qryLoaded;

  // Set up kind of destination

  DestinationKind := skText;

end;

dtmMain.Execute;
