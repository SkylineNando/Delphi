Demo ‚ConnectionDefs’

Illustrate the different ways, how a connection can be defined.

 

Property ‘ConnectionDefFileAutoLoad’

Loading the ConnectionDefFile automatically when a connection definition is the first time used.

ADManager.ConnectionDefFileAutoLoad:= True;

 

Property ‘ConnectionDefFileLoaded’

Using ConnectionDefFileLoaded property to check if connection definition file is loaded.

if not ADManager.ConnectionDefFileLoaded then

…

 

a.)    Create connection definition on fly

Create/ Add Connection Definition:

with ADManager.ConnectionDefs.AddConnectionDef do begin

  Name := 'MyDefNew';

  DriverID := 'MSAcc';

  Database := '$(DEHOME)\DB\Data\DEDemo.mdb';  // using properties

  Params.Add('Tracing=True');                  // using Params

  AsBoolean['ReadOnly'] := True;               // using AsXXX properties

  // mark persistent to save this connection definition to file later

  MarkPersistent;

end;

// to make new definition persistent call following:

ADManager.ConnectionDefs.Save;

Connect to RDBMS:

DEConnection1.Connected:= True;    

Delete Connection Definition:

if ADManager.ConnectionDefs.FindConnectionDef(‘MyDef’) <> nil then

  ADManager.ConnectionDefs.ConnectionDefByName(‘MyDef’).Delete;

 

b.)    Load connection definition file

Load ‘Connection Definition File’ explicitly with the method ‘LoadConnectionDefFile’

  ADManager.ConnectionDefFileAutoLoad := False;

  ADManager.LoadConnectionDefFile;

The file is searched by default beside the application (in the same path). If the one is not found it takes the location of the ‘Connection Definition File’ from the registry.

c.)     Set connection definition file name explicitly (with/ without path)

Explicitly setting the ConnectionDef file name, this file will be searched beginning the path of your application.

ADManager.ConnectionDefFileName := 'MyTest.ini’;

d.)    Add connection definition

It’s like point a). The difference is that you may create connection definition after loading other connection defs from file.

e.)    Clone Connection definition.

procedure ...

var

  oConnectionDef: IDEStanConnectionDef;

begin

  oConnectionDef := ADManager.ConnectionDefs.ConnectionDefByName(‘MyDef’);

  with ADManager.ConnectionDefs.AddConnectionDef do begin

    Name := ‘NewDef’;

    // Clone connection definition assigning the parameters

    Params.AddStrings(oConnectionDef.Params);

  end;

end;
