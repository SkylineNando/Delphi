1                                     Transactions

This demo shows simple ways to work with transactions.

1.)     Setting up transaction isolation level.

with dmlMainBase.dbMain do begin

    TxOptions.Isolation := xiReadCommitted;

2.)     Controlling Auto commit mode

    TxOptions.AutoCommit := False;

3.)     Starting transaction

    StartTransaction;

    try

    ...
