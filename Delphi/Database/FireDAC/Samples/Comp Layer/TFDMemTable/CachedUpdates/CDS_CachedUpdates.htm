1         Cached Updates

The demo shows simple using a TFDMemTable.

Looking at the Form you can see the set of components:

a)       TFDMemTable

b)       TFDTableAdapter

c)       TFDCommand

How this set is work.

TFDCommand uses TFDConnection to connect to the desired database. In the command text of TFDCommand there is a SQL command for selecting data from DB: ‘select * from {id Orders}’

TFDTableAdapter in the common case can manipulate with the several TFDCommand components and is a mediator between TFDCommand and TFDMemTable.

TFDMemTable retrieves data from DB through TFDTableAdapter and TFDCommand.
