Demo Name VersionInfo
Classes demonstrated

Custom components: TVersionInfo,TVersionInfoLegalCopyrightLabel, TVersionInfoFileDescriptionLabel, TVersionInfoProductNameLabel, TVersionInfoFileVersionLabel

An About box showing detailed version information based on component properties

Localization of version information
Description To run the example, complete the following steps:

- Load VersionInfoGroup.bpg.

- Build VersionInfo60.bpl.

- Install DCLVersionInfo60.bpl to install the version components.

- Build and run VersionInfoSample.exe to see a sample About box.

- Build TestProject and check properties in Explorer.
