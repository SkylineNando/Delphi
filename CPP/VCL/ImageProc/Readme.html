This demo shows the use of TImage and related components to manipulate characteristics of images including contrast, negatives, and arithmetic operations. Follow these steps to try it out:

Steps

Result
1 - File | Open (boots.bmp)
2 - Edit | Convert to Grayscale
3 - Edit | Flip vertically 		The boots are now inverted and black and white.
4 - Select | Image 2
5 - File | Open boots.bmp
6 - Edit | Convert to Grayscale
7 - Select | Image 3
8 - Edit | Arithmetic Operations
9 - Select the following options:
     First Image: Image 1
     Operator: Average
     Second Image: Image 2
     Result: Image 3
10 - Click ok
The inverted and normal boots are superimposed.

1 - File | Open (civil-war.bmp)
2 - Filter | Median (Repeat this step several times)
3 - Edit | Negative
4 - Filter | Detect Edges

The soldiers' camp photo has fewer spots, is negative, and has the "line drawing" effect.

1 - File | Open (minature_rockers.bmp)
2 - Edit | Convert to Grayscale
3 - Light | Apply Threshold
4 - Select | Image 2
5 - Repeat steps 1 and 2 
6 - Light | Manual adjust, move contrast up, brightness down
7 - Select | Image 3
8 - Edit | Arithmetic Operations
9 - Select the following options:
     First Image: Image 1
     Operator: Subtract 
     Second Image: Image 2
     Result: Image 3

The chairs show a mixture of high contrast and gray tones. Try out the result with the other Operator choices.

1 - File | Open (ViewOfWashington.bmp)
2 - Edit | Convert to grayscale
3 - Light | Manual Adjust...
4 - Move Contrast down and click ok
5 - Light | Auto contrast

The resulting image from step 4 has its contrast adjusted.
