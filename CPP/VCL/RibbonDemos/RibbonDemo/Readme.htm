Description This demo demonstrates the main features of the new Ribbon controls. Such interfaces are designed to facilitate user access to application’s menu options. The architecture behind Ribbon Controls is very simple. From a Ribbon control you are able to add a Tab (which contains groups). Each of these groups contains buttons with customized appearance. Additionally, the Ribbon control includes the Quick Access Toolbar and the Application Menu.
