Description This demo is a system with functionality to add, modify, browse and report on orders, customers, parts, and items. Also demonstrates how to access a help file from a C++Builder application.
Uses TIBDatabase component.

This example requires that Interbase XE be installed. Make sure the server is running before you run the example.
